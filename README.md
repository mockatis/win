SEVPN_install.bat - в указанной директории должен находиться самораспаковывающийся архив с файлами конфигурации клиента и исполняемыми файлами SoftEther VPN Client.
admin_eac_request - заготовка для повышения прав.
findstr_usage - заготовка для поиска по маске в процессе выполнения скрипта, аналог grep в *nix
nbtns_disable - отключение nbtns, уязвимого для спуфинга
